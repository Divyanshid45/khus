# News Aggregator

A simple Node.js + Express app that fetches news using the NewsAPI.

## Features

- Search by keyword
- Display top 5 articles
- Clean UI with EJS and CSS

## Getting Started

1. Clone this repo
2. Run `npm install`
3. Create `.env` with your NewsAPI key
4. Run `node app.js`
5. Visit `http://localhost:3000`
